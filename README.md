# User Management Task

This guide provides step-by-step instructions to run UserManagementTask MVC project.

## Prerequisites
- .NET 9 SDK
- Visual Studio 2022: Ensure it is updated to a version that supports .NET 9 or use the appropriate preview
- NuGet Package Feeds
- Database: SQLServer (Ensure the connection string is set in appsettings.json)

## How to run

- clone the repo from this url (https://github.com/abdallahimam/UserManagementTask.git)
- Open the Solution
- Restore NuGet Packages: Go to Build > Restore NuGet Packages, or right-click the solution in Solution Explorer and select Restore NuGet Packages.
- Run the project
